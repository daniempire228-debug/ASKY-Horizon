## Installation
To install the ASKY Horizon demo app, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/daniempire228-debug/asky-horizon.git
   ```
2. Navigate into the directory:
   ```bash
   cd asky-horizon
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To run the demo app, execute the following command:
```bash
npm start
```
